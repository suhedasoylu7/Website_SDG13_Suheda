# Climate Action Hackathon 2025 – Landing Page

## About
This is a landing page for the 'Climate Action Hackathon 2025' focused on Urban Heat & Flood Resilience, connected to SDG 13: Climate Action. It gives information about the event, speakers, tracks, resources, registration, sponsors, and social contacts.

The page is made using only HTML5 and CSS — no JavaScript — and is responsive and accessible.

---

## Features
Hero Section: Event name, theme, and "Register Now" button.  
About & SDG 13: Explains the purpose of the event and its connection to climate action.  
Schedule: Table with key dates and activities.  
Speakers & Mentors: Cards with images, names, and roles.  
Tracks / Challenges: Heat Mitigation, Flood Early-Warning, Green Infrastructure, Community Apps.  
Resources: Links to datasets and APIs useful for participants.  
Registration Form: Name, email, role, and project idea.  
Sponsors & Partners: Logos displayed in a grid.  
Contact & Socials: Email and social media icons (Facebook, Instagram, TikTok).  
FAQ: Expandable answers with visible styling.
